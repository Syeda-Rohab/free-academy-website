CHAPTER 3: VARIABLES (var/let/const)
============================================================

3.1 Variable Kya Hai?
   Variable ek container hai jo data store karta hai.

3.2 var Keyword (Old Way - Ab use na karein):
   var oldVariable = "I am var";
   Value: I am var
   Problem: Function-scoped, redeclare kar sakte hain

3.3 let Keyword (Modern Way):
   let modernVariable = "I am let";
   Value: I am let
   After change: Changed!
   Feature: Block-scoped, change kar sakte hain

3.4 const Keyword (For Constants):
   const constantValue = "Cannot Change";
   Value: Cannot Change
   Feature: Block-scoped, change NAHI kar sakte

3.5 Variable Naming Rules:
   ‚úì let userName = "Alice";  (camelCase)
   ‚úì let user_age = 25;        (snake_case)
   ‚úì let $price = 100;         ($ allowed)
   ‚úì let _private = "secret";  (_ allowed)
   ‚úó let 123name = "test";    (number se start nahi)
   ‚úó let user-name = "test";  (- allowed nahi)

3.6 Best Practice:
   - let aur const use karein
   - var avoid karein
   - Descriptive names dein
   - camelCase follow karein

‚úì Variables Complete!

CHAPTER 4: DATA TYPES
============================================================

4.1 Primitive Data Types (7 types):

   1. STRING - Text ke liye
      let str = "Hello JavaScript";
      Value: Hello JavaScript
      Type: string

   2. NUMBER - Numbers ke liye (integer & decimal)
      let num = 42;
      let float = 3.14;
      Values: 42 and 3.14
      Type: number

   3. BOOLEAN - True/False ke liye
      let bool = true;
      Value: true
      Type: boolean

   4. NULL - Intentionally empty value
      let empty = null;
      Value: null
      Type: object

   5. UNDEFINED - Value assign nahi kiya
      let notAssigned;
      Value: undefined
      Type: undefined

   6. SYMBOL - Unique identifier ke liye
      let sym = Symbol("unique");
      Type: symbol

   7. BIGINT - Very large numbers ke liye
      let big = 9007199254740991n;
      Type: bigint

4.2 Reference Data Types:

   1. ARRAY - Multiple values store karne ke liye
      let arr = [1, 2, 3, 4, 5];
      Value: [ 1, 2, 3, 4, 5 ]
      Type: object

   2. OBJECT - Key-value pairs store karne ke liye
      let obj = { name: "Alice", age: 25 };
      Value: { name: 'Alice', age: 25 }
      Type: object

   3. FUNCTION - Reusable code block
      Type: function

4.3 Type Conversion:
   String to Number: Number("42") = 42
   Number to String: String(42) = 42
   To Boolean: Boolean(1) = true

CHAPTER 5: OPERATORS & EXPRESSIONS
============================================================

5.1 Arithmetic Operators:
   Addition (+):      10 + 3 = 13
   Subtraction (-):   10 - 3 = 7
   Multiplication (*): 10 * 3 = 30
   Division (/):      10 / 3 = 3.3333333333333335
   Modulus (%):       10 % 3 = 1
   Exponentiation (**): 10 ** 2 = 100
   Increment (++):   x = 10 ‚Üí x++ = 11
   Decrement (--):   x = 11 ‚Üí x-- = 10

5.2 Assignment Operators:
   a = 5
   a += 3  ‚Üí a = 8
   a -= 2  ‚Üí a = 6
   a *= 2  ‚Üí a = 12
   a /= 2  ‚Üí a = 6

5.3 Comparison Operators:
   Equal (==):       5 == '5' ‚Üí true
   Strict Equal (===): 5 === '5' ‚Üí false
   Not Equal (!=):   5 != '5' ‚Üí false
   Strict Not (!==): 5 !== '5' ‚Üí true
   Greater Than (>): 10 > 5  ‚Üí true
   Less Than (<):    10 < 5  ‚Üí false
   >= (Greater or Equal): true
   <= (Less or Equal):    false

5.4 Logical Operators:
   AND (&&): true && false ‚Üí false
   OR (||):  true || false ‚Üí true
   NOT (!):  !true         ‚Üí false

5.5 Ternary Operator:
   let age = 20;
   let status = age >= 18 ? "Adult" : "Minor";
   Result: Adult

5.6 Expression Examples:
   (10 + 5) * 2 - 3 = 27

‚úì Operators Complete!

CHAPTER 8: ARRAYS
============================================================

8.1 Array Create karna:
   let fruits = ["Apple", "Banana", "Cherry"];
   let numbers = [1, 2, 3, 4, 5];
   let mixed = [1, "two", true, null];

8.2 Array Access karna:
   fruits[0] ‚Üí Apple
   fruits[1] ‚Üí Banana
   fruits.length ‚Üí 3

8.3 Array Methods - Add/Remove:
   fruits.push("Date")     - Add to end
   Result: [ 'Apple', 'Banana', 'Cherry', 'Date' ]
   fruits.pop()            - Remove from end
   Result: [ 'Apple', 'Banana', 'Cherry' ]
   fruits.unshift("Apricot") - Add to beginning
   Result: [ 'Apricot', 'Apple', 'Banana', 'Cherry' ]
   fruits.shift()            - Remove from beginning
   Result: [ 'Apple', 'Banana', 'Cherry' ]

8.4 Array Methods - Search:
   fruits.indexOf("Banana") ‚Üí 1
   fruits.includes("Cherry") ‚Üí true

8.5 Array Methods - Transform:
   fruits.join(", ") ‚Üí Apple, Banana, Cherry
   fruits.slice(1, 3) ‚Üí [ 'Banana', 'Cherry' ]

8.6 Array Iteration Methods:

   forEach - Har element pe execute:
   nums.forEach(num => console.log(num));
   Output:
      1
      2
      3
      4
      5

   map - Transform array:
   nums.map(num => num * 2) ‚Üí [ 2, 4, 6, 8, 10 ]

   filter - Filter elements:
   nums.filter(num => num % 2 === 0) ‚Üí [ 2, 4 ]

   find - First match dhundo:
   nums.find(num => num > 3) ‚Üí 4

   reduce - Single value mein convert:
   nums.reduce((acc, num) => acc + num, 0) ‚Üí 15

CHAPTER 11: EVENTS
============================================================

11.1 Events Kya Hain?
    User actions jo JavaScript code trigger karte hain

11.2 Event Listeners Lagana:
    element.addEventListener('click', handler)
    element.addEventListener('submit', handler)
    element.addEventListener('keydown', handler)

11.3 Common Events:
    Mouse Events:
    - click: User click kare
    - dblclick: Double click kare
    - mouseover: Mouse upar aaye
    - mouseout: Mouse hatae
    - mousemove: Mouse move kare

    Keyboard Events:
    - keydown: Key press kare
    - keyup: Key release kare
    - keypress: Key press ho rahi ho

    Form Events:
    - submit: Form submit ho
    - change: Input value change ho
    - input: Input mein typing ho
    - focus: Input focus ho
    - blur: Input blur ho

    Page Events:
    - load: Page load ho
    - scroll: User scroll kare
    - resize: Window resize ho

11.4 Event Object:
    element.addEventListener('click', function(event) {
      event.target      // Clicked element
      event.type        // Event type (click, keydown, etc.)
      event.preventDefault()  // Default action roko
      event.stopPropagation() // Bubbling roko
    });

11.5 Practical Example:
    // Form Validation
    const form = document.querySelector('form');
    form.addEventListener('submit', function(e) {
      e.preventDefault();  // Page reload roko
      const email = document.getElementById('email').value;
      if (!email.includes('@')) {
        alert('Invalid email!');
      }
    });

11.6 Event Bubbling & Capturing:
    Bubbling: Child ‚Üí Parent (default)
    Capturing: Parent ‚Üí Child
    addEventListener('click', handler, true)  // Capturing
    addEventListener('click', handler, false) // Bubbling

‚úì Events Complete!
